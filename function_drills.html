for (initialization; condition; increment/decrement) { // Code to be executed in each iteration } Initialization: This step is used to initialize a variable before the loop starts. It typically involves declaring a loop counter variable and assigning an initial value to it. For example, let i = 0; initializes the loop counter variable i with an initial value of 0. Condition: This step defines the condition that is checked before each iteration. If the condition evaluates to true, the loop continues executing. If it evaluates to false, the loop terminates. For example, i < 5 is a condition that checks if the loop counter i is less than 5. Increment/Decrement: This step updates the loop counter variable after each iteration. It modifies the value of the loop counter to ensure that the loop eventually terminates. For example, i++ increments the loop counter i by 1 after each iteration. Code Block: This is the block of code that is executed in each iteration of the loop. It contains the statements or actions you want to perform repeatedly. Example: const array = [1, 2, 3, 4, 5]; for (let i = 0; i < array.length; i++) { console.log(array[i]); } The i variable is initialized with 0. The condition i < array.length checks if the loop counter i is less than the length of the array. As long as it's true, the loop continues. The code block console.log(array[i]); is executed in each iteration, printing the value of the array element at index i. After each iteration, the loop counter i is incremented by 1 (i++). As a result, the loop will iterate over the array from index 0 to the last index, printing each element to the console.
